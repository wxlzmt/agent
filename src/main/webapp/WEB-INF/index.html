调用路径：#(ctx)/q Method：POST Content-Type: application/x-www-form-urlencoded; charset=UTF-8

请求参数：键以此种前缀开始param. header. 分别代表请求的源参数和源头部

params

参数 必填 说明
url 是 请求访问的URL地址
method 否 取值:GET或POST;默认值:GET
paramEncoding 否 请求参数的编码,以防中文乱码.默认值:UTF-8
responseEncoding 否 HTTP响应正文的编码.默认值:UTF-8
enctype 否 当method=POST时有效;取值: #(com.zkext.agent.controller.MainController::ENCTYPE_FORM_URLENCODED) 或 #(com.zkext.agent.controller.MainController::ENCTYPE_RAW) ; 默认值=#(com.zkext.agent.controller.MainController::ENCTYPE_FORM_URLENCODED)
#(com.zkext.agent.controller.MainController::PREFIX_PARAM)raw 否 当enctype=#(com.zkext.agent.controller.MainController::ENCTYPE_RAW)时,需要传递本参数作为POST的body
#(com.zkext.agent.controller.MainController::PREFIX_HEADER)Content-type 否 强烈建议传递此参数
